### 2. **[HIGH PRIORITY]** Genetic Algorithm with Crossover
Implement a GA that can escape local optima:

```python
# Population: 100 solutions
# Initialize with:
#   - saspav baseline
#   - Random perturbations of baseline
#   - Lattice-based solutions
# 
# Crossover: For each N, take trees from parent A or B
# Mutation: Small position/rotation perturbations
# Selection: Tournament selection, keep top 50%
# 
# Run for 1000 generations
```

**Why this can work:** GA explores multiple basins of attraction simultaneously and can combine good parts from different solutions.

### 3. **[MEDIUM PRIORITY]** Basin Hopping
Random large perturbation followed by local optimization:

```python
# For each N:
# 1. Start from baseline
# 2. Apply large random perturbation (move trees by 0.1-0.5 units)
# 3. Run local optimization (bbox3 or SA)
# 4. If better, keep; else revert
# 5. Repeat 100 times
```

**Why this can work:** Explores different basins of attraction that local optimization alone cannot reach.
