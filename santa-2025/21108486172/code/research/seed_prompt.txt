## Key Insight from Research

The No-Fit Polygon (NFP) approach is the standard technique for 2D polygon packing:
1. NFP represents all valid relative positions between two polygons
2. Placing a tree on the NFP boundary guarantees touching but not overlapping
3. This turns the continuous geometry problem into discrete candidate positions
4. Enables efficient exploration of the configuration space

**The baseline was likely found using NFP-based optimization. We need to use the same technique but explore different basins.**
