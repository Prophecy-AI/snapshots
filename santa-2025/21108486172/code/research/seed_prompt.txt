## Recommended Approaches (Priority Order)

### 1. **[HIGHEST PRIORITY] Gradual Constraint Tightening from Overlapping Configs**
The overlapping configs (67.727) represent a BETTER basin of attraction. The approach:
1. Start from the overlapping configuration for each N value
2. Identify the overlapping pairs and their overlap areas
3. Gradually tighten constraints by:
   - Moving overlapping trees apart by small amounts
   - Rotating trees to reduce overlap
   - Using a penalty function that balances score vs. overlap
4. Gradually increase the overlap penalty until overlaps are eliminated

**Why this might work:** The overlapping configs are in a BETTER basin - we want to stay in that basin while eliminating overlaps.

### 2. **[HIGH PRIORITY] Constructive Heuristics (Bottom-Left Placement)**
Build packings tree-by-tree with intelligent placement:
1. Use No-Fit Polygon (NFP) to find all valid placements for each tree
2. Place trees one at a time using bottom-left strategy
3. Use different orderings (by size, by angle, random)
4. This explores a DIFFERENT solution space than SA

**Reference:** The research shows bottom-left heuristics with NFP are effective for irregular packing.

### 3. **[HIGH PRIORITY] Tabu Search with Large-Scale Moves**
Break out of local optimum using:
1. Tabu list to prevent reversing recent moves
2. Large-scale "shake" moves - remove subset of trees and re-insert
3. Guided Local Search (GLS) with penalties on "culprit" trees
4. This forces exploration of new configurations

### 4. **[MEDIUM PRIORITY] Analyze Overlapping Configs in Detail**
Before trying to fix overlaps, understand WHY they're better:
1. What angle patterns do they use?
2. What position patterns do they use?
3. Can we extract valid substructures?
4. Are there N values where the overlapping config is actually valid?

### 5. **[MEDIUM PRIORITY] Hybrid GA-LP Approach**
Combine genetic algorithm with local search:
1. Encode packing as chromosome (order + orientation)
2. Run GA operators (crossover, mutation)
3. Apply fast local search to each offspring
4. GA provides global diversification, local search provides exploitation
