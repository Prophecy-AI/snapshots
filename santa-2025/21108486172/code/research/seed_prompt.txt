### 1. **[HIGHEST PRIORITY]** Fix N=9 Overlap
The baseline has an overlap at N=9. This MUST be fixed first.

### 2. **[HIGH PRIORITY]** Implement BackPacking Kernel (crodoc)
The BackPacking kernel uses a backward iteration strategy:
- Start from N=200 and work backward
- When a configuration at N is worse than N+1 with one tree removed, copy from N+1
- This propagates good patterns from larger to smaller N

### 3. **[HIGH PRIORITY]** Higher-Parameter Greedy Backtracking
The blueshyy kernel uses:
- DEPTH=10, BEAM=10, MAX_STATES=4000
- RAND_TRIES=3, RAND_K=20
- Multiple PASSES with parallel processing

Our experiment 013 used beam_width=5, max_states=500 - 8x fewer states!

### 4. **[MEDIUM PRIORITY]** Two-Tree Unit Optimization
Instead of simple grid tiling, optimize the unit cell geometry:
- Create pairs of trees with opposite orientations (0° and 180°)
- Optimize the relative position within the pair
- Then tile the optimized pairs

### 5. **[MEDIUM PRIORITY]** Exhaustive Search for Small N
For N=2-10 (which contribute 53.8% of the gap):
- Use exhaustive search with fine grid
- Try all angle combinations
- These are computationally tractable
