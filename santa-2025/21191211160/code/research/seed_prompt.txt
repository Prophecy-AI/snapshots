## Recommended Approaches (Priority Order)

### 1. **[HIGHEST PRIORITY] Implement "Eazy Optimizer" C++ with Square Potential Gradient**
The jazivxt/eazy-optimizer kernel uses a fundamentally different approach:
- **Log-barrier derivatives**: d/dx [-ln(L-x) - ln(L+x)] = 1/(L-x) - 1/(L+x)
- **Square Calculus Pressure**: Pushes trees toward center using gradient descent
- **Complex Orbital Moves**: Uses complex number rotation for tree movement
- **Elastic Pulse**: Periodic squeeze and relax cycles

This is NOT just another SA - it's a gradient-based approach that may find different basins.

**Implementation:**
```bash
# Compile the eazy optimizer
g++ -O3 -fopenmp -o eazy eazy.cpp
./eazy
```
