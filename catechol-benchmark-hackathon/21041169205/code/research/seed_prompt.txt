## Recommended Approaches

### PRIORITY 1: Nearest Neighbor Blending (Instead of Global Mean)
The evaluator's top recommendation. For outlier solvents, blend toward NEAREST training solvent's predictions instead of global mean. This preserves chemical similarity.

Implementation:
```python
def get_nearest_training_mean(self, test_solvent, training_solvents, solvent_predictions):
    """Get mean prediction for k nearest training solvents."""
    test_features = SPANGE_DF.loc[test_solvent].values
    distances = []
    for train_solvent in training_solvents:
        train_features = SPANGE_DF.loc[train_solvent].values
        dist = np.linalg.norm(test_features - train_features)
        distances.append((train_solvent, dist))
    
    distances.sort(key=lambda x: x[1])
    nearest = distances[:3]  # k=3 nearest neighbors
    
    # Return mean of nearest solvents' predictions
    nearest_preds = [solvent_predictions[s] for s, _ in nearest]
    return np.mean(nearest_preds, axis=0)
```
